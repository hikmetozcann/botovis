# Botovis Dokümantasyon

Laravel uygulamalarınıza yapay zeka destekli veritabanı asistanı ekleyen Botovis'in Türkçe dokümantasyonuna hoş geldiniz.

## İçindekiler

| Doküman | Açıklama |
|---------|----------|
| [Kurulum](installation.md) | Gereksinimler, adım adım kurulum, sorun giderme |
| [Yapılandırma](configuration.md) | Tüm ayarlar, ortam değişkenleri, sürücü yapılandırması |
| [Güvenlik](security.md) | Kimlik doğrulama, roller, RBAC, izinler, şema filtreleme |
| [Araçlar](tools.md) | Yerleşik 8 araç, özel araç oluşturma rehberi |
| [Widget](widget.md) | Web bileşeni, öznitelikler, API, tema, React/Vue entegrasyonu |
| [API Referansı](api-reference.md) | HTTP uç noktaları, SSE olayları, istek/yanıt formatları |
| [Artisan Komutları](artisan-commands.md) | Keşif ve CLI sohbet komutları |
| [Mimari](architecture.md) | Paket yapısı, agent döngüsü, sürücüler, genişletme noktaları |

## Hızlı Bağlantılar

- **İlk kurulum mu?** → [Kurulum Rehberi](installation.md) ile başlayın
- **Ayarları yapılandırmak mı?** → [Yapılandırma](configuration.md) sayfasına bakın
- **Özel araç mı yazacaksınız?** → [Araçlar](tools.md#özel-araç-oluşturma) bölümüne gidin
- **Widget'ı özelleştirmek mi?** → [Widget](widget.md) rehberine göz atın
- **Güvenlik endişeleri mi?** → [Güvenlik](security.md) katmanlarını inceleyin
